Some libraries for geospatial analysis

Python libraries:

1- GDAL/OGR: Geospatial Data Abstraction Library GDAL is a low-level library for reading and writing geospatial data formats. It is used for tasks such as format translation, projections, and data manipulation.

2- Fiona: It is a Python library for reading and writing vector data, such as shape files. It is built on GDAL and provides a more Python-based interface for working with geospatial data.

3- GeoPandas: It is an open source library that extends the capabilities of Pandas data frames to handle geospatial data. It simplifies tasks such as reading, writing, and analyzing geospatial data.

4- Shapely: It is a Python library for geometric operations on geospatial objects. It allows you to perform operations such as intersection, union and caching on geometric shapes.

5- Basemap: It is a Matplotlib toolkit for creating static 2D maps. It has now been deprecated in favor of Cartopy and other libraries.

6- Cartopy: It is a library for mapping projections and visualizing geospatial data within Matplotlib. It is used to create maps and work with geospatial data in Python.

7- PyProj: It is a Python interface to the PROJ library, which is used for mapping projections and coordinate transformations.

R libraries:

1- sf: The sf package in R is designed to work with simple feature (SF data), which is a unified way to represent and process geospatial vector data.

2- raster: It is used in the R language to work with raster data, such as satellite images and grid datasets.

3- leaflet for R: It is an R package that allows you to create interactive web maps using the Leaflet JavaScript library.

4- sp: The sp package is an old package for processing spatial data in R. It is often used with rgdal to read and write vector data.

JavaScript libraries:

1- Leaflet: It is a popular open source JavaScript library for creating interactive maps in web applications.

2- Mapbox GL JS: It is a JavaScript library from Mapbox for creating custom interactive maps for web and mobile applications.
